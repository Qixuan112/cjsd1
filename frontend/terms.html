1. 服务说明

KiraAI 插件商店（以下简称"本平台"）是一个开源插件分发平台，致力于为用户提供便捷的插件浏览、下载和管理服务。

主要功能包括：

  • 插件浏览与搜索：用户可以通过分类、关键词等方式查找所需插件
  • 插件下载：提供插件的安装包下载服务
  • 插件管理：开发者可以提交、更新和管理自己发布的插件
  • 用户认证：通过 GitHub OAuth 实现安全的用户身份验证

2. 用户责任

使用本平台服务时，您同意遵守以下规定：

  • 遵守法律法规：不得利用本平台从事任何违法违规活动，包括但不限于传播恶意软件、侵犯他人知识产权、发布虚假信息等
  • 账号安全：妥善保管您的 GitHub 账号信息，对账号下的所有行为负责
  • 内容真实性：确保提交的插件信息真实、准确、完整
  • 不滥用资源：不得使用自动化工具恶意爬取平台数据，不得进行任何可能影响平台正常运行的操作
  • 尊重他人：在平台内的所有互动中保持礼貌，不得发布侮辱、诽谤或骚扰性内容

3. 插件发布规范

开发者提交的插件需要满足以下要求：

  • 必要文件：插件必须包含完整的 manifest.json 文件，其中应包含插件名称、版本、描述、作者等基本信息
  • 信息真实：插件描述、功能介绍等信息必须真实准确，不得虚假宣传
  • 安全性：插件不得包含恶意代码、病毒、木马或其他可能危害用户设备安全的内容
  • 知识产权：插件不得侵犯任何第三方的知识产权，包括但不限于版权、商标权、专利权等
  • 隐私保护：插件收集用户数据时必须明确告知用户，并获得用户同意
  • 开源许可：建议插件使用明确的开源许可证（如 MIT、GPL、Apache 等）

审核流程：所有提交的插件都需要经过平台审核，审核通过后才能上架。平台保留拒绝或下架任何不符合规范的插件的权利。
